Difference to AI and Deep Learning

https://blogs.nvidia.com/blog/2016/07/29/whats-difference-artificial-intelligence-machine-learning-deep-learning-ai/
Artificial Intelligence Started in the 50s Rule-based No learning Example TicTacToe - unbeatable Hype during the last years, new hardware, faster processing Machine Learning Started in the 80s Subcategory of AI - ML is always AI, AI is not always ML Learn from existing data to make predictions about unknown data No handcoded rules necessary Deep Learning New trend in ML Inspired by the brain Many small neurons ("stupid/simple") in a network Surprise Extremely good performance in image/language recognition Unknown what's going on.
